A. Persistent Left Sidebar

Filters always visible in a fixed-width sidebar. Products in a 3-column grid beside it. Desktop only — collapses to current horizontal layout on mobile.

Pros

  • Products + filters visible simultaneously
  • Common e-commerce pattern (Amazon, Myntra)
  • Checkbox multi-select feels natural

Cons

  • Sidebar takes ~220px, products get fewer columns on medium screens
  • Lots of filter groups = long scrolling sidebar

B. Collapsible Icon Sidebar

Thin icon strip on the left that expands into full sidebar on hover/click. Products get the full 4-column grid by default. Best of both worlds.

Pros

  • Products get max space when not filtering
  • Feels clean and modern
  • Sidebar available on demand

Cons

  • More complex interaction — users may not discover it
  • Icons alone can be unclear

C. Keep Horizontal Pills + Slide-Over Drawer (Current, Refined)

Keep the top category pills as quick-access, but slim them to ONE row (categories only). All other filters move into the "Filters" slide-over drawer. Products get full width.

Pros

  • Products always get maximum space (4 columns)
  • Already built — minimal change needed
  • Mobile-first (works same on all devices)

Cons

  • Filters hidden behind a button (extra click)
  • Can't see filters + products at same time
