A — Show & manage the default

The admin always shows the currently-live hero (even the default), with the same Replace / Crop tools as regular slides. Uploading a new image replaces it permanently.

Pros

  • What you see in admin = what shows on site
  • Can crop focal point of default image

Cons

  • Slightly more logic to track "default" vs DB slides

B — Informational notice + upload

Show a small "currently showing default image" notice at the top of the Hero tab. Clicking "Add slide" replaces it. No way to crop the default — but once you upload your own, full controls appear.

Pros

  • Simpler — no special default handling
  • Clear nudge to upload your own image

Cons

  • Can't crop the default image's focal point
  • Default stays until you upload
